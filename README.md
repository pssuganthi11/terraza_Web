#  Tourconn – Tourism Package Application 
<p>Tourconn is a smart tourism platform that helps users discover, compare, and book travel packages while enabling vendors and admins to manage everything seamlessly in one place.</p>
---

## 🟡 Guest User UI (Updated)

### 🎯 Goal
Encourage users to explore travel packages with clear understanding of trip types and detailed itinerary.


### 📌 Requirements
- Visually rich homepage with hero banner (images/video)
- Smart search bar (destination, date, travelers)

### 🔹 Category Section (NEW)
- Display trip categories:
  - Group Trip 👥
  - Family Trip 👨‍👩‍👧
  - Solo Trip 🧍
  - Honeymoon 💑
  - Adventure 🏔️
- Each category as clickable cards with icons/images
- On click → filter packages based on category

---

### 🔹 Filters
- Country
- Budget
- Duration
- Travel type

---

### 🔹 Featured Packages
- Card layout with:
  - Image
  - Destination
  - Price
  - Duration
  - Category tag

---

###  Package Details Preview (ENHANCED)
Each package should clearly show:

####  Day-wise Plan (Itinerary)
- Day 1: Arrival + sightseeing
- Day 2: Activities (e.g., trekking, city tour)
- Day 3: Checkout / return

####  What’s Included
- Hotel stay 🏨
- Food 🍽️
- Transport 🚗
- Guide 🧭

#### ❌ What’s Not Included
- Personal expenses
- Extra activities
- Travel insurance

#### ⚠️ Do’s & Don’ts
- Do:
  - Follow schedule
  - Carry ID proof
- Don’t:
  - Miss reporting time
  - Carry restricted items

---

### 🔹 Popular Destinations
- Image grid with clickable locations

---

### 🔹 Authentication
- Login / Signup modal

---

### 🔹 Testimonials & Footer
- Reviews and contact info

---


### ⚙️ Goal
Help users quickly understand:
- Type of trip
- What they will experience
- What is included/excluded

And encourage them to sign up for booking.
---

##  Logged-in User UI

###  Goal
Provide a smooth, personalized booking experience.



###  Requirements
- Dashboard with:
  - Welcome message
  - Summary cards (trips, bookings)
- Advanced search with filters & sorting
- Package details page:
  - Image gallery
  - Full itinerary
  - Reviews
  - "Book Now" button
- Booking page:
  - Traveler form
  - Date picker
  - Pricing summary
- My Trips section (upcoming, completed, cancelled)
- Profile section
- Notifications


---

##  Vendor UI

###  Goal
Help vendors manage packages and bookings efficiently.



###  Requirements
- Dashboard with stats:
  - Total packages
  - Bookings
  - Revenue
- Add/Edit package form - multi-step maybe:
  - Details
  - Pricing
  - Itinerary
  - Image upload
- Package listing (edit/delete, status)
- Booking requests panel (accept/reject)
- Revenue dashboard (charts)
- Profile management

---

##  Admin UI

###  Goal
Provide full system control and monitoring.

###  Requirements
- Dashboard with KPIs:
  - Users
  - Vendors
  - Revenue
- User & vendor management:
  - Approve / Block / Delete
- Package monitoring & approval
- Revenue analytics (charts, filters)
- Reports (PDF/CSV download)
- System alerts
