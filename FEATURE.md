# 📌 Tourconn – Tourism Package Application

Tourconn is a smart tourism platform that helps users discover, compare, and book travel packages while enabling vendors and admins to manage everything seamlessly.

---

## 👀 Guest User Features

- 🔍 Browse and explore travel packages  
- 🎯 Filter packages by:  
  - Destination  
  - Budget  
  - Duration  
  - Travel type (Group, Solo, Family, etc.)  
- 📄 View basic package details:  
  - Overview  
  - Price  
  - Duration
  - Day-wise itinerary  
- 🧭 View trip categories (Group, Solo, Family)  
- 🌍 Explore popular destinations  
- 🔐 Signup / Login to access full features  

---

## 👤 Logged-in User Features

- 📄 View detailed package information:   
  - Inclusions & exclusions  
  - Do’s & Don’ts  
- 🧳 Book travel packages  
- 📅 Select travel date and number of travelers  
- 📂 View **My Trips**:  
  - Upcoming trips  
  - Completed trips  
  - Cancelled trips  
- 🌦️ View **Climate Insights** for booked trips:  
  - Temperature  
  - Weather condition  
  - Rain probability
  - it may be api & historical report   
- 💡 Smart travel tips based on weather  
- 🔐 Secure authentication & session handling  

---

## 🏢 Vendor Features

- ➕ Add new travel packages  
- ✏️ Edit existing packages  
- 🗑️ Delete packages  
- 📋 View all added packages  
- 🔄 Track package status:  
  - Pending  
  - Approved  
- 📦 View bookings for their packages  
- 👤 Manage vendor profile  

---

## 🛠️ Admin Features

- 👥 View all users  
- 🏢 View all vendors  
- ✅ Approve or reject travel packages  
- 🚫 Block / Unblock users  
- 🚫 Block / Unblock vendors  

---

## 🎯 Project Goal

- Provide a seamless travel booking experience  
- Help users clearly understand trip details  
- Enable vendors to manage packages easily  
- Give admins full control over the platform
